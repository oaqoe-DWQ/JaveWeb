# Javeweb-
JavaWeb环境需要jdk、eclipse和tomcat,先安装jdk并配置环境变量，再安装eclipse，再解压tomcat，在eclipe里安装tomcat，这里我们先安装jdk和eclipse。

------------------------------------------------------------------------------------------------------------------
	这种方式的步骤是：先安装并配置jdk--->双击压缩包--->选择下载安装合适的版本

注意：所有工具的路径中最好不要有中文或空格

----------------------------------------------------------------------------------------------------
本次课程步骤：
1.装jdk
2.配置jdk环境变量
	JAVA_HOME
	Path
	CLASSPATH
3.安装eclipse
